The Project

The Dart Linux Project aims to be a fully systemd-free distro, completely using BusyBox and some coreutils from GNU, like Bash.

It has a simple service language, it is mostly fbcon-based terminal, but you can choose what WM to install. For now we are avoiding X11 and Wayland (They're RAM Eaters)

Since this project is almost lightweight to fit in an embedded system, we are starting to add i686 (x86) support in our ports, and coming soon, ARM support.
